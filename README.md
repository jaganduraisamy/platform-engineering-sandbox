# platform-engineering-sandbox
Welcome to my personal cloud-native sandbox! This repository serves as a hands-on laboratory where I experiment with Infrastructure as Code (IaC), GitOps workflows, Kubernetes orchestration, CI/CD pipelines, and enterprise-grade observability stacks.

The goal of this repo is to build, break, fix, and document modern platform engineering patterns.

---

## 🛠️ Tech Stack & Ecosystem

| Category | Tools Explored |
| :--- | :--- |
| **Orchestration & Compute** | Kubernetes (K8s), Minikube / Kind, Helm |
| **Infrastructure as Code** | Terraform, OpenTofu |
| **GitOps & Delivery** | ArgoCD, GitHub Actions |
| **Observability & Monitoring** | Prometheus, Grafana, OpenTelemetry, Loki |
| **Service Mesh & Networking** | Istio / Linkerd, Ingress-NGINX |
| **Security & Secrets** | HashiCorp Vault, Sealed Secrets |
